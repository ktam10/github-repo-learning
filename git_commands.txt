GIT and GITHUB

git add . : to add all the files from local sys to stage
git status : to check status of files
git commit -m "commit message" : to add all the files from stage to be tracked on Github
rm testing.py : to remove a file

  git remote add origin "URL" : to add the created repo on git
  git remote -v : Will show fetch and push origins
  git push origin master : Will throw an error due to auth issues
  git remote set-url origin "URL with secret" : Created a token from github and then executed this command
  git remote -v : the URL for fetch and push has changed
  git push origin master : Push the changes on github
  git config --global user.name "your-username" : Configured git username
